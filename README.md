## 🚀 Overview

SkyAI is a prototype of an **AI-powered flight price comparison and prediction platform** built as part of a Technology Commercialization assignment. It demonstrates how artificial intelligence can be applied to solve real problems faced by airline travelers.

## 🎯 Problem Statement

Travelers waste hours manually checking multiple websites for the best flight prices, often missing deals or booking at the wrong time, resulting in overpaying by 20–40%.

## 💡 Solution

An intelligent platform that:
- **Compares** flight prices across 50+ airlines in real-time
- **Tracks** price changes continuously with smart alerts
- **Predicts** future price trends using AI/ML models
- **Recommends** the best time and flight to book

## 🖥️ Pages

## 📊 Key Features

- ✅ Real-time flight comparison
- 🔔 Price drop alerts
- 🤖 AI booking recommendations (Buy / Wait / Avoid)
- 📈 7-day price forecast
- 📉 30-day historical price chart
- 🏷️ Airline scoring (AI Score out of 100)
